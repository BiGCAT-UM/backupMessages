503 Service Unavailable

This page indicates you got redirected because something is not working. Please read the instructions and pointers below about the status of our BiGCaT services.

6 January 2020

Because of a security incident at the Maastricht University, the network is isolated from the outside world and several BiGCaT servers are offline. At the moment, it is unclear when these services will be restored. You can follow one or more of our Twitter accounts to receive direct updates.

Working services:

WikiPathways website and data download

The WikiPathways website is online and can be used. Pathway data can be downloaded as well .

BridgeDb webservice

The BridgeDb webservices are online and can be used. Data included are the latest Gene, Protein, Metabolite and Interaction mappings. However, the Swagger/OpenAPI interactive documentation is not available.

NOT Working services:

PathVisio.org

The PathVisio website is unreachable, along with the data.pathvisio.org website hosting the BridgeDb identifier mapping databases. You can download PathVisio from our Github release page. For metabolite mappings , Figshare has an archived version of a recent mapping file. This is also the case for interactions mappings . The BridgeDb webservice includes these latest files, as well as Gene+Protein mappings.

WikiPathways SPARQL endpoint

The WikiPathways SPARQL endpoint is also unreachable (WikiPathways.org is not affected). We currently do not have an alternative, but one option is to download the RDF data and load this in a local Virtuoso or Blazegraph installation..

BridgeDb.org website

The BridgeDb website is unreachable. Along with the data.bridgedb.org website hosting the BridgeDb identifier mapping databases. For metabolite mappings , Figshare has an archived version of a recent mapping file. This is also the case for interactions mappings. The BridgeDb webservice includes these latest files, as well as Gene+Protein mappings.

ArrayAnalysis.org website and service

The ArrayAnalysis.org website is unreachable. Results from analysis started shortly before the attack will be kept on the server.
